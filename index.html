Google Sheets Setup Required

To connect live data, publish each sheet tab as CSV:

  1. Open your Google Sheet
  2. Go to File → Share → Publish to web
  3. Select each Qlik tab individually, choose CSV format, and click Publish
  4. Copy each published URL
  5. Open this HTML file in a text editor and paste the URLs into the SHEET_URLS config at the top of the <script> section
